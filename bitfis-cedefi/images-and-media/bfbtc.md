### Using bfBTC in DeFi

Multiple ecosystem projects have integrated bfBTC, allowing users to stake bfBTC and earn additional rewards. You can find DeFi projects that support bfBTC staking on the [**BitFi Ecosystem**](https://app.bitfi.one/ecosystem) page.



### Multi-Chain Support

BitFi BTC has been launched on ETH, BSC, BASE, BTR, HEMI, CORE, GOAT, and AILAYER (deposits and withdrawals are currently not supported on AILAYER). Users can stake BTC assets on these chains and receive bfBTC.

In the future, BitFi will enable cross-chain transfers of bfBTC using the LayerZero bridge



### Comprehensive Infrastructure

BitFi supports integration with multiple wallets.

In addition, RedStone provides price feed services for bfBTC, enabling developers to access real-time bfBTC price data.
